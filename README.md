# 📊 Dashboard de Análisis de Datos en Power BI con DAX

## Descripción general (Overview)

Este proyecto consiste en el desarrollo de un dashboard interactivo en Power BI orientado al análisis y visualización de datos para apoyar la toma de decisiones empresariales. Se utilizaron técnicas de modelado de datos, creación de medidas mediante DAX (Data Analysis Expressions) y diseño de visualizaciones dinámicas para transformar los datos en información clara y accionable.

---

## Conjunto de datos (Dataset)

El conjunto de datos utilizado contiene información empresarial que fue importada y preparada dentro de Power BI para su posterior análisis.

Las actividades realizadas incluyen:

* Importación y conexión de datos.
* Revisión y transformación de tablas.
* Validación de la calidad de los datos.
* Creación de relaciones entre tablas.
* Preparación del modelo de datos para el análisis.

---

## Herramientas y Tecnologías (Tools)

Las tecnologías utilizadas en este proyecto fueron:

* **Power BI Desktop:** Desarrollo del dashboard, modelado de datos y diseño de visualizaciones interactivas.
* **DAX (Data Analysis Expressions):** Creación de medidas, cálculos personalizados y KPIs.
* **Power Query:** Limpieza, transformación y preparación de los datos antes del análisis.

---

## Pasos del proceso (Project Workflow)

### 1. Importación de datos

Se cargaron los datos en Power BI y se realizó una revisión inicial de las tablas y su estructura.

### 2. Transformación de datos con Power Query

Se realizaron procesos de limpieza, modificación de formatos y preparación de los datos para garantizar la consistencia del modelo.

### 3. Modelado de datos

Se establecieron relaciones entre las diferentes tablas y se diseñó un modelo eficiente para optimizar el análisis.

### 4. Creación de medidas con DAX

Se desarrollaron medidas personalizadas para calcular métricas de negocio, indicadores clave de rendimiento (KPIs) y análisis comparativos.

Ejemplos de cálculos realizados:

* Ventas totales.
* Crecimiento porcentual.
* Comparación entre períodos.
* Promedios y acumulados.
* Indicadores de rendimiento.

### 5. Diseño del Dashboard

Se construyeron visualizaciones interactivas utilizando:

* Tarjetas de KPIs.
* Gráficos de barras y líneas.
* Segmentadores (slicers) para filtrar información.
* Tablas y matrices.
* Análisis comparativos y tendencias.

---

## Dashboard

El dashboard permite explorar los datos de manera dinámica mediante filtros e interacciones entre visualizaciones, facilitando la identificación de tendencias, comportamientos y métricas clave del negocio.

---

## Resultados (Results)

El proyecto permitió convertir datos en información útil para la toma de decisiones mediante:

* Creación de un modelo de datos estructurado.
* Desarrollo de medidas avanzadas con DAX.
* Visualización de KPIs estratégicos.
* Identificación de tendencias y patrones relevantes.

---

## Cómo ejecutar el proyecto (How to Run)

1. Descargar o clonar el repositorio.
2. Abrir el archivo `.pbix` en Power BI Desktop.
3. Actualizar las fuentes de datos si es necesario.
4. Explorar el dashboard utilizando los filtros y segmentadores disponibles.
5. Revisar las medidas DAX y el modelo de datos desde las vistas de Modelo y Datos de Power BI.
